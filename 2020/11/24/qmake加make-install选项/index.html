qmake加make install选项

Posted on 2020-11-24

一般的qmake生成的Makefile是没有
make install的

方法：
在.pro中做文章
比如你要安装libEbookDataBase.so*到目录 /usr/local/lib
.pro中加入

‘’’
inst.files += libEbookDataBase.so*
inst.path = /usr/local/lib
INSTALLS += inst
‘’’

‘’’
qmake
就可以
make
make install
‘’’
